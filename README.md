
# Getting started


https://horoiwa.github.io/ProjectBoilerPlate/


- Local environment

`cd project && poetry run --no-root`

`poetry run python -V`

`poetry run pytest`

- Container environment

`sudo ./cmd.sh up`

`sudo ./cmd.sh login`

`cd project && python -V && pytest`


- Product environment

`sudo ./cmd.sh product`
